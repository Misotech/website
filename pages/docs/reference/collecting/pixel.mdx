Используется для:

1. Контроля количества показов и пост-вью аналитики рекламных материалов
2. Установки на сторонние сайты без установки JS кода
3. Выполнения cookie-sync стандартным подходом, т.е. при помощи редиректа пикселя

## Endpoint

```
GET  https://домен/img/:name/:projectId
```

можно без projectId

```
GET  https://домен/pixel/:name
```

помимо `pixel` еще работают `img` и `gif`

### Ограничения

Размер данных ограничен допустимым размером url в браузере.




export default withDoc({...meta})(({children}) => <>{children}</>)
