Основные компоненты

1. **Band framework** - framework для разработки компонентов Rockstat на Python
2. **Band director** - сервис для создания и управления контейнерами
3. **Rockme.ts** - фреймворк для разработки микросервисов Rockstat на TypeScript
4. **Front** - сервис для получения данных, а также управления жизненным циклом входящих запросов
5. **ChWriter** - сервис управляющий схемой данных в Clickhouse, а также записи входящих данных учитывая структуру БД

SDK

1. **Web SDK**  - браузерная библиотека для отслеживания поведения посетителей на сайтах. Объединяет события в сессии, отслеживает активность, клики и взаимодействия с формами.

Стандартные обогатители данных

1. **UserAgent parser** - сервис извлечения полезной информации из User-Agent браузера
2. **MaxMind geolocation** - сервис геолокации по ip адресу на основе базы данных MaxMind
3. **Sypex geolocation service** - сервис геолокации по ip адресу на основе базы данных SypexGeo

Другие сервисы:

1. **Send Mixpanel **- сервис стриминга данных в MixPanel, как пример создания аналогичных сервисов
2. **Heavyload** - сервис приема больших файлов
3. **TgHelloBot** - пример реализации чатбота

Стороннее ПО адаптированное для Rockstat

1. **Grafana** - инструмент для построение дашбордов на основе данных из ClickHouse и других источников. Дополнения: драйвер Сlickhouse  и расширенный набор виджетов
2. **Anaconda** - аналитическая сборка Python включающая содержащая библиотеки для анализа данных. Дополнения: дополнительные часто используемые пакеты, автозапуск Jupyter.
3. **Theia IDE** - специальная сборка персональной среды разработки под Rockstat. Поддерживается разработка на Python, TypeScript и Javascript. Проброшен удобный терминал (zsh + zprezto) с сервера. Предусмотрен запуск сервисов для отладки.
4. **ClickHouse Proxy** - посредник при работе с ClickHouse. Контролирует нагрузку на СУБД, обрабатывает лимиты, ставит запросы в очередь, поддерживает CORS

Остальное ПО,

1. **Netdata** - мониторинг сервера
2. **Docker** - микросервисы
3. **logspout** - стриминг логов из docker в сторонние сервисы
4. **ClickHouse** - открытая колоночная СУБД от Yandex. Имеет очень высокую скорость работы, а данные хранит в сжатом виде
5. **Redis** - Key-value хранилище работающее в оперативной памяти
6. **Nginx** - http сервер, используется как балансировщик
7. **LetsEncrypt - SSL** сертификаты
