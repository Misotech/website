# О платформе Rockstat

Rockstat - это платформа, позволяющая построить собственную CDP (Customer Data Platform) для управления жизненным циклом пользователей,
не затратив на это много времени.
Платформа включает в себя: WebSDK, сервис сбора данных, сервис записи и управления ClickHouse, систему оркестровки, 
frameworks для Python и TypeScript, сервисы геокодинга, разбора UserAgent, средства разработки, инструменты обработки и визуализации данных,
СУБД ClickHouse, Redis, систему управления и множество других необходимых составляющих.

<Tip>
  В данный момент Rockstat проходит стадию Alpha, это
  допускает возможность изменения API компонентов платформы и наличие сбоев.
</Tip>
