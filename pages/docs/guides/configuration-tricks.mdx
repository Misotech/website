# Хитрости конфигурации Rockstat

## Theia IDE

#### Длина строки pycodestyle

Добавить в корень проекта файл `config.cfg` c содержимым:

```ini
[pycodestyle]
max-line-length = 250
```

## Префикс для переменных в cookies пользователей

Rockstat присваивает пользователю внутренний айдишник и при использовании Web-SDK для трекинга событий пишет этот айдишник в wildcard cookies и local storage. По дефолту переменная называется `uid`.

При установке в большой проект есть вероятность, что переменная `uid` уже используется для внутренних нужд и её занимать нельзя.

Чтобы это исправить, можно задать кастомное имя для это переменной. Вот как:

Добавляем в папку `/project/config/custom/front` файл `config.yaml` и пишем в него что-то типа того:

```ini
identify:
  param: rst-uid
```

`rst-uid` — это название переменной, которая будет в cookies и local storage и в которой будет лежать айдишник Rockstat. Обзываем так, чтобы она не конфликтовала с внутренними.
